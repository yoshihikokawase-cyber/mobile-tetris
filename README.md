# Mobile Tetris

HTML / CSS / JavaScript のみで動く、PC・スマホ完全対応版テトリスです。

---

## ファイル構成

```
mobile-tetris/
├── index.html   # メイン HTML
├── style.css    # スタイル（レスポンシブ対応）
├── script.js    # ゲームロジック完全版
└── README.md    # このファイル
```

---

## 動作確認手順

1. `index.html` をブラウザで開く（ファイルをダブルクリック or ドラッグ＆ドロップ）
2. 「START」ボタンをクリックしてゲーム開始
3. PC・スマホ両方で動作確認可能

**ローカルサーバーで確認する場合（推奨）:**
```bash
cd mobile-tetris
python -m http.server 8080
# → http://localhost:8080 を開く
```

## 操作方法

### PC キーボード

| キー | 操作 |
|------|------|
| `←` / `→` | 左右移動 |
| `↓` | ソフトドロップ（1マス落下） |
| `↑` | 回転 |
| `Space` | ハードドロップ（即座に落下・固定） |
| `P` | 一時停止 / 再開 |
| `R` | リスタート |

### スマホ タッチ操作

| 操作 | 動作 |
|------|------|
| ◀ LEFT ボタン | 左移動 |
| ▶ RIGHT ボタン | 右移動 |
| ↑ ROTATE ボタン | 回転 |
| ▼ DOWN ボタン | ソフトドロップ |
| ⬇ DROP ボタン | ハードドロップ |
| 画面タップ | 回転 |
| 左右スワイプ | 左右移動 |
| 下スワイプ | ソフトドロップ |
| 上スワイプ | ハードドロップ |

## スコア計算

| 消去ライン数 | 基本点 |
|-------------|--------|
| 1ライン | 100 × レベル |
| 2ライン | 300 × レベル |
| 3ライン | 500 × レベル |
| 4ライン（テトリス） | 800 × レベル |

- ソフトドロップ: 1マス = +1点
- ハードドロップ: 1マス = +2点
